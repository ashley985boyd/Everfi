# Contribution Guidelines

We welcome contributions to the Everfi Expense Tracker project! Please follow the guidelines below.

## How to Contribute

1. Fork the repository.
2. Clone your forked repository.
3. Create a new branch (`feature/your-feature-name`).
4. Make your changes.
5. Commit your changes with clear messages.
6. Push to your fork.
7. Open a pull request.

## Reporting Issues

- Use the issue tracker to report bugs.
- Provide clear descriptions and steps to reproduce.
- Include screenshots if applicable.

## Suggesting Features

- Open a feature request issue.
- Clearly describe the feature and its benefits.
- Provide examples if possible.

## Code of Conduct

- Be respectful and inclusive.
- No harassment or discrimination.
- Maintain professionalism in discussions.

## Review Process

- All pull requests are reviewed by maintainers.
- Feedback must be addressed before merging.
- Approved PRs will be merged into the main branch.
